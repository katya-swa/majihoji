# Majihoji

メンズ化粧品専用のアイテム共有サイトです。

URL https://majihoji.herokuapp.com/

要件定義/画面設計まとめページ

https://quip.com/yKhFApgCgaVM



# 機能一覧

* ログイン・ログアウト機能
* 投稿作成/更新/削除機能
* 投稿一覧機能
* 投稿検索機能
* 投稿コメント機能(コメント作成/削除)
* 画像アップロード機能(Carrierwave/S3へのアップロード)
* ページネーション機能

# 主な環境

* Ruby 2.5.1
* Rails 5.2.3
* devise
* Git/Github
* ransack
* Bootstrap
* Font-awesome
* PostgreSQL
* heroku
